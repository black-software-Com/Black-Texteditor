# Black-Texteditor v1.0
<hr>

**Installing On Linux & MacOS**
``` sh
git clone https://github.com/black-software-Com/Black-Texteditor

cd Black-Texteditor

python black.py
```
<br>

*Installing On Windows*
``` sh
scoop install git or choco install git

git clone https://github.com/black-software-Com/Black-Texteditor

cd Black-Texteditor
```
